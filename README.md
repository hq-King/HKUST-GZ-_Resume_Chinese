二、修改说明 (Changelog for HUST Version)
------------------------------------------------------------------------
为了适配香港科技大学(广州)的风格，并降低使用门槛，做了以下改动：

1. 配色调整：
   - 将原本的“华科红 (HUST_Red)”替换为“科广蓝 (HKUST_GZ)”。
   - 颜色代码调整为 RGB: 0,71,122，符合 HUST 官方视觉识别系统。

2. 结构优化 (重要)：
   - 废弃了原模板需要依赖外部图片 (`header.png`/`footer.png`) 来显示页眉页脚色块的方式。
   - 改用 TikZ 代码直接绘制红色背景条。这意味着你不需要懂 PS，也不用制作背景图，代码会自动生成高清的蓝色横条，方便调整高度和颜色。

3. 字体兼容性：
   - 优化了 `settings.tex` 中的字体设置。增加了对 Overleaf 和普通 TeX 发行版的兼容性检测，默认使用 Fandol 系列字体（宋体/黑体），避免了“缺少 SweiSpring 字体”导致的编译报错。

4. 本地化内容：
   - 预置了“人工智能学域”、“信息枢纽”等示例。
   - 替换了相关的校徽引用逻辑。

三、使用方法 (Usage)
------------------------------------------------------------------------
1. 编译环境：
   - 必须使用 **XeLaTeX** 编译器 (因为涉及到中文和字体处理)。
   - 推荐平台：Overleaf (在线) 或 VS Code + LaTeX Workshop (本地)。

2. 图片准备 (在 images 文件夹中):
   - avatar.png:       你的证件照 (推荐比例 3:4)。
   - UST-GZ_white_tc_L_PNG:    科广长条校名 Logo (建议白色文字或透明底，显示在页眉左上角)。
   - in_UST-GZ_4C_en-C_PNG:   科广校徽 (用于正文中心的透明水印)。
   *注：原模板中的 header.png 和 footer.png 已不再需要。*

四、常见问题 (Q&A)
------------------------------------------------------------------------
Q1: 图标显示不出来，或者想换图标？
A1: 模板使用了 FontAwesome5 宏包。
    请查阅图标代码表：https://fontawesome.com/v5/search?m=free
    例如 \faGithub, \faEnvelope, \faPython 等。

Q2: 怎么改栏目名字（比如把“项目经历”改成“实习经历”）？
A2: 在 main.tex 中找到 \section{...}。
    例如：\section{\makebox...{\faLaptopCode}}\quad 项目与实习经历}
    修改大括号最后的文字即可。

Q3: 觉得颜色不好看，想换？
A3: 打开 settings.tex，找到 \definecolor{HKUST_GZ}{RGB}{0,71,122}。
    修改里面的 RGB 数值即可。例如深蓝色是 {0, 51, 102}。
